# Dotide.rb

Ruby library for Dotide API

[![Build Status](https://travis-ci.org/dotide/dotide.rb.png?branch=master)](https://travis-ci.org/dotide/dotide.rb)

## Installation

Add this line to your application's Gemfile:

    gem 'dotide'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dotide
